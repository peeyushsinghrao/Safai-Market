Target Users

- Kirana Shops
- Grocery Stores
- General Stores
- Wholesale Stores
- Retail Shops
- Multi-Branch Businesses

---

Product Vision

Build a production-ready store management platform that feels as simple as WhatsApp, PhonePe and Blinkit while providing enterprise-level inventory, billing, reporting and multi-shop capabilities.

The application must be mobile-first but fully optimized for tablets and desktop systems.

The system should allow shop owners to manage products, inventory, customers, bills, expenses, suppliers, devices and reports from a single platform.

---

Core Product Goals

- Extremely simple UI
- Fast billing workflow
- Multi-shop architecture
- Offline-first operation
- Barcode-enabled inventory
- GST-ready billing
- Multi-device support
- Future branch expansion support
- Production-grade scalability

Multi-Shop System

Architecture:

- Products
- Customers
- Bills
- Reports
- Expenses

Success Criteria

The application should allow a shop owner to:

- Create a shop
- Add products quickly
- Scan barcodes
- Bill customers rapidly
- Track inventory
- Manage multiple stores
- Work offline
- Generate GST invoices
- Print and share bills
- Manage devices
- Scale to multiple branches

while keeping the user experience extremely simple and intuitive.
